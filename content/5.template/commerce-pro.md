# Commerce Pro

A premium, modern e-commerce storefront template built with Nuxt 3.

## Installation

1. Clean install dependencies:
   ```bash
   pnpm install
   ```
2. Copy environmental variables:
   ```bash
   cp .env.example .env
   ```
3. Run in dev mode:
   ```bash
   pnpm dev
   ```
