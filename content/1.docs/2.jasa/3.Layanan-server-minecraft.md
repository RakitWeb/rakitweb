## 2. Spesifikasi Teknis Server

### 2.1. Kebutuhan Hardware & RAM

Server Minecraft memiliki kebutuhan resource yang sangat spesifik dan berbeda dari hosting web pada umumnya. Kebutuhan RAM ditentukan oleh beberapa faktor utama: jumlah pemain, jumlah mod/plugin, ukuran dunia, dan versi Minecraft yang digunakan.

**Rumus Perhitungan RAM (Rule of Thumb)**:

`RAM yang Dibutuhkan = 1 GB (base) + (0.25 GB × jumlah pemain) + (0.5 GB × jumlah plugin ÷ 10) + (1 GB × jumlah mod ÷ 20)`

**Tabel Kebutuhan RAM Berdasarkan Skenario**:

| RAM | Kapasitas Pemain (Vanilla) | Kapasitas Pemain (Modded) | Penggunaan Ideal |
|-----|---------------------------|---------------------------|------------------|
| 1 GB | 1-4 pemain | Tidak Direkomendasikan | Testing / single-player |
| 2 GB | 5-10 pemain | Tidak Direkomendasikan | Server pribadi kecil |
| 3 GB | 10-20 pemain | 3-8 pemain | Server komunitas kecil |
| 4 GB | 15-20 pemain | 10-15 pemain | Light modpack / server sedang |
| 6 GB | 25-30 pemain | 15-20 pemain | Medium modpack |
| 8 GB+ | 40+ pemain | 20+ pemain | Heavy modpack / server besar |

**Spesifikasi CPU yang Direkomendasikan**:

Minecraft server sangat bergantung pada **single-core performance** CPU. Server (mulai versi 1.14) memang menggunakan core tambahan untuk operasi lain, tetapi biasanya maksimal tiga core yang terpakai. Hyperthreading (HT) tidak memberikan keuntungan signifikan untuk Minecraft.

RakitWeb merekomendasikan:
- **CPU dengan clock speed tinggi (3.5GHz+)**: untuk performa tick rate (TPS) yang stabil
- **Prosesor AMD Ryzen atau Intel Core series**: untuk keseimbangan performa dan efisiensi
- **NVMe SSD**: untuk loading chunk yang lebih cepat dibandingkan SSD biasa

**Kebutuhan Java Runtime**:

Setiap server Minecraft Java Edition memerlukan Java Runtime Environment (JRE) dengan versi yang sesuai:
- Versi 1.17 → JRE 16
- Versi 1.18 → JRE 17
- Versi 1.20.5 ke atas → JRE 21

### 2.3. Keamanan & Perlindungan DDoS

Server Minecraft publik adalah target utama serangan DDoS. RakitWeb menerapkan perlindungan berlapis:

- **Cloudflare-protected network**: Perlindungan DDoS di level jaringan
- **XDP/eBPF-based firewall**: Filtering paket malicious di level kernel untuk Java Edition
- **Rate Limiting & Connection Control**: Plugin seperti DDoSGuard untuk membatasi koneksi simultan dari IP yang sama
- **Enterprise-grade DDoS Protection**: Dengan deteksi berbasis machine learning

### 2.4. Panel Kontrol & Manajemen

Setiap server Minecraft RakitWeb dilengkapi dengan panel kontrol modern yang memudahkan pengelolaan:
